Winows7 的 Python3.7.2 环境下安装 zhpy1.7.4 的操作

操作步骤：

（1）下载 Python 的 Python3.7.2 Win32 版本并安装到 C:\Program Files (x86)\Python\Python37-32；

（2）将 zhpy1.7.4 的压缩包文件解包到目录 C:\Program Files (x86)\Python\Python37-32
（3）保持互联网的网络连接状态；
（4）运行windows7的 cmd，进入 DOS 命令窗口；
（5）cd C:\Program Files (x86)\Python\Python37-32
（6）dir > 安装前目录内容.txt
（7）python setup.py install > 安装过程记录.txt
（8）查 安装过程记录.txt ,正常安装成功。

安装操作顺利完成。但是，原程序里含有与 Python3 不兼容的 Python2 命令，主要是 print 和 except，需按照出错信息的提示去修改相关的程序源码。出错信息是：

其中，上述后五项错误与前面的五项雷同，其实，它们都出自于 zhpy 目录下的相同名文件。所以，修改 zhpy 目录下的相同名文件的Python版本兼容性语句的隐形瑕疵。然后，将修改后的新文件复制到上述相关目录 ...\build\dbist.win32\egg\zhpy 和 ...\lib\site-packages\zhpy-17.7.4-py3.7.egg\zhpy，重复前述（5）之后的操作步骤。
